# 素人种草运营工作台

本地网页工作台，用来管理 40+ 兼职/账号的每日内容派发、素材调度、爆款提权、图文/视频交付、抖音核对和数据回填。

## 启动

双击根目录的 `启动素人工作台.command`，或在终端运行：

```bash
cd /Users/licc/Desktop/素人系统/app
npm install
npm run start
```

默认地址：

```text
http://127.0.0.1:5174
```

## 数据和素材

- 结构化数据：`/Users/licc/Desktop/素人系统/data/souren.sqlite`
- 本地备份：`/Users/licc/Desktop/素人系统/data/backups`
- 真实案例素材：`/Users/licc/Desktop/素人系统/素材库/真实案例`
- 网页端只保存路径和状态，不把图片、视频写进数据库。
- 新建案例后，系统会自动创建 `00-原始素材`、`01-已筛选素材`、`02-生成补充`、`03-交付给兼职`、`04-发布回收`。
- 顶部可以手动创建本地 JSON 备份；导入备份前系统会自动创建一份 `before-import` 备份，降低误覆盖风险。

## 核心模块

- 今日中控台：待生成、待选择、待交付、待核对、异常账号。
- 数据复盘：账号表现、内容类型进度、生命周期分布、重点关注账号、最近回填数据。
- 运营建议：异常账号会给出下一步动作，例如补排期、处理素材缺口、加爆款提权或优先核对。
- 排期规划：按未来 7/14/30/60 天查看所有账号内容安排，并直接处理生成、交付、派发和核对动作。
- 案例库：新建、批量导入、搜索筛选、删除数据库记录。
- 内容种子：维护素人种草和日常养号的本地生成模板。
- 爆款模板：录入爆款结构、来源链接、适合/禁用人设关键词，并批量生成账号人设化版本。
- 本地交付包：生成可直接通过电脑微信发送的文案、图片、视频任务文件夹。
- 本地交付包会包含素材顺序清单，方便按顺序拖进微信发送。
- 图片任务：预留 Image v2 接入，`IMAGE_API_KEY` 为空时进入 `waiting_key`，同时生成本地图片任务说明。
- 核对队列：打开抖音链接后人工/半自动回填播放、点赞、评论、粉丝数据。

## 环境变量

在 `app/.env` 里预留：

```env
IMAGE_API_KEY=
LLM_API_KEY=
```

当前 key 为空时，文案、排期、交付包、核对和数据复盘仍可正常使用。
