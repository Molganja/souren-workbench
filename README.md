# 素人种草运营工作台

本地网页工作台，用来管理 40+ 兼职/账号的每日内容派发、素材调度、爆款提权、图文/视频交付、抖音核对和数据回填。

## 启动

双击根目录的 `启动素人工作台.command`，或在终端运行：

```bash
cd /Users/licc/Desktop/素人系统/app
npm install
npm run start
```

默认地址：

```text
http://127.0.0.1:5174
```

同一局域网里的工作人员要访问时，在主机 `app/.env` 设置：

```env
SOUREN_HOST=0.0.0.0
SOUREN_ACCESS_CODE=给工作人员的访问码
```

重启后进入「系统配置」复制局域网网址，发给工作人员即可。工作人员只打开网页、输入访问码、处理对接和回填；图片生成、爆款分析和系统数据仍在主机本地完成。远端工作人员不会看到打开主机本地目录的按钮，所有日常动作都在网页内完成。

## 桌面客户端

开发机上可以直接用桌面窗口运行：

```bash
cd /Users/licc/Desktop/素人系统/app
npm run client
```

生成 Mac 桌面应用目录：

```bash
cd /Users/licc/Desktop/素人系统/app
npm run client:mac
```

输出目录是 `app/release/`。客户端包不会带真实 `.env`、数据库或素材；运行时会在本机自己的工作数据目录里创建数据和素材库。Mac `.app` 会用 Electron 内置 Node 启动本地后端并打开工作台窗口，不需要工作人员手动启动服务。

## 数据和素材

- 结构化数据：`/Users/licc/Desktop/素人系统/data/souren.sqlite`
- 真实案例素材：`/Users/licc/Desktop/素人系统/素材库/真实案例`
- 网页端只保存路径和状态，不把图片、视频写进数据库。
- 新建案例后，系统会自动创建 `00-原始素材`、`01-已筛选素材`、`02-生成补充`、`03-交付给兼职`、`04-发布回收`。
- 新建案例默认随机生成人设，并自动生成近期排期；工作人员只需要填对接咨询/负责人和抖音主页/作品链接，其他信息可后续编辑。

## 工作人员每天只做两件事

1. 打开「今日中控台」，看今天每个兼职该发什么，生成候选、选择、查看交付内容，然后在网页内复制文案、下载图片/视频，用电脑微信发送。
2. 打开「爆款链接」，把看到的抖音爆款作品链接贴进系统；后续提文案、拆结构、判断适合账号和生成同类型内容由系统使用者处理。

完整每日操作步骤见：`日常运营SOP.md`。

## 核心模块

- 今日中控台：按对接咨询/负责人汇总今日任务，明确每条内容发给哪个兼职、由谁对接；同时显示今日链路复盘，可一键复制工作简报，并可查看待生成、待选择、素材阻塞、待交付、待核对、待爆款分析、图片任务、剪辑任务和异常账号。
- 运营建议：异常账号会给出下一步动作，例如补排期、处理素材缺口、加爆款提权或优先核对。
- 排期规划：按未来 7/14/30/60 天查看所有账号内容安排，并直接处理生成、交付、派发和核对动作。
- 案例库：新建、批量导入、搜索筛选、删除案例；删除时会同步删除本地案例素材目录。
- 爆款链接：工作人员只贴链接；完成内容提取和结构分析后再批量生成账号人设化版本。
- 网页交付面板：交付文件仍落在本地，但工作人员在网页内复制文案、预览/下载图片和视频，不需要每天打开交付目录。
- 视频/口播交付会额外展示口播字幕、剪辑要求、源素材下载和成片保存路径。
- 图片任务：预留图片生成接口，接口未接入时显示“待接入图片接口”，仍可生成本地图片任务说明和可复制提示词；接口接入后可在网页内生成并下载图片。
- 剪辑任务：生成剪辑任务单，可在首页复制任务单并标记状态；本机管理端才显示打开目录入口。
- 核对队列：打开抖音链接后人工/半自动回填播放、点赞、评论、粉丝数据，并可复制核对清单。

## 自动化边界

- 系统可以独立完成：排期、候选稿、交付内容、素材路径管理、图片提示词、剪辑任务、核对队列和数据记录。
- 系统可以接入图片接口后完成：图片生成任务。文案、排期、交付和核对不依赖接口。
- 抖音核对默认人工回填：工作人员打开链接，看发布时间、内容、播放、点赞、评论、粉丝，再在系统回填。
- 浏览器自动读取只能作为增强：如果后续做客户端或浏览器插件，可以尝试读取当前页面上的公开数据；读不到、登录失效、页面改版时，仍回到人工回填。
- 不依赖当前聊天窗口时，客户端仍能运行；但“替你看抖音、临时判断、跨网页操作”这类能力需要浏览器协同或人工完成。

## 环境变量

在 `app/.env` 里预留：

```env
PORT=5174
SOUREN_HOST=127.0.0.1
SOUREN_ACCESS_CODE=
IMAGE_API_KEY=
IMAGE_API_URL=
IMAGE_MODEL=image-v2
IMAGE_SIZE=1024x1024
```

图片生成需要同时填写 `IMAGE_API_KEY` 和 `IMAGE_API_URL`；未填写时，图片任务仍可复制提示词，不影响文案、排期、交付和核对。

当前图片接口未接入时，文案、排期、网页交付和核对流程仍可正常使用。
