# 素人种草运营工作台 v1

本地网页工作台，用于管理兼职账号、三类内容排期、候选稿、素材扫描、微信交付包、Image 任务预留和抖音核对队列。

## 启动

最简单方式：双击项目根目录的：

```text
启动素人工作台.command
```

它会自动检查环境、安装依赖、构建前端、启动本地服务并打开浏览器。

命令行开发方式：

```bash
cd /Users/licc/Desktop/素人系统/app
npm install
npm run dev
```

- 前端：`http://127.0.0.1:5173`
- 后端/API：`http://127.0.0.1:5174`

生产构建后也可以只跑后端：

```bash
npm run build
npm run start
```

打开：`http://127.0.0.1:5174`

## Image / LLM Key

`app/.env` 里预留：

```env
IMAGE_API_KEY=
LLM_API_KEY=
```

当前 v1 会创建 Image 任务和 prompt。`IMAGE_API_KEY` 为空时，任务状态为 `waiting_key`，不会影响其他流程。

## 当前闭环

1. 工作人员打开「今日中控台」，只看今天该处理的账号和动作。
2. 新建案例并生成本地案例目录。
   - 新建时会随机生成人设，并自动生成近期排期。
   - 工作人员默认只需要填对接咨询/负责人和抖音主页/作品链接；昵称、人设、抖音号都可以后续编辑。
   - 也可以在案例库使用「批量导入」，一次录入多名兼职/账号，并自动生成排期。
3. 把原始素材放进 `00-原始素材/`，点击扫描素材。
4. 系统生成排期槽位，对当天或近期槽位生成 3 条候选稿。
5. 工作人员选择一条并锁定。
6. 生成微信交付包。
   - 如果没有可用素材，任务会进入「素材阻塞」，不会误进入可交付。
   - 素材阻塞任务可以直接复制补素材说明、打开素材目录，补齐后重新生成交付包。
   - 如果是视频内容，可以从排期或案例详情创建「剪辑任务」，系统会生成本地剪辑任务单。
7. 工作人员复制文案、打开交付文件夹，用电脑微信发给兼职。
8. 标记派发、兼职已汇报。
9. 打开抖音链接核对并回填状态。
10. 工作人员在「爆款链接」里只粘贴抖音爆款作品链接；青豆提取、拆结构和生成同类型内容由 Codex/AI 处理。

## 新增实用操作

- 「排期规划」页可以按未来 7/14/30/60 天、状态、内容类型和账号关键词查看全局排期，并直接处理生成、交付、派发和核对动作。
- 「今日中控台」会按对接咨询/负责人汇总今日任务，工作人员能直接知道每条内容该发给哪个兼职、由谁对接。
- 「今日链路复盘」按状态串起待生成、待选择、已锁定、素材阻塞、可交付、已派发、已汇报、已核对，并支持复制今日工作简报。
- 「复制AI工作包」会在 `data/operator-packets/` 生成 Markdown，并把内容复制到剪贴板，里面包含 Codex 下一步优先级、今日链路、待青豆分析、图片/剪辑任务、异常账号和本地路径。
- 「请本地AI建议」会优先调用 `claude`、`clude`、`claude-code`，也可以通过 `LOCAL_CLAUDE_COMMAND` 指定命令；结果会保存到 `data/ai-consults/`，首页会展示最近记录并能打开原文件。如果命令未安装，系统会保存调用记录，不影响其他功能。
- 「今日中控台」会直接显示素材阻塞、待爆款分析、Image 任务、剪辑任务、待核对任务。
- 「今日中控台」支持一键准备今日交付：生成候选、锁定稳妥稿、生成交付包；缺素材的任务会进入素材阻塞，不会计入可交付。
- 案例库支持批量导入兼职/账号，适合一次录入 40+ 账号。
- 案例库支持按关键词、阶段、健康状态搜索筛选，也可以删除误建案例。
- 案例详情里可以直接「打开素材目录」。
- 案例详情里可以编辑案例信息，也可以手动新增临时内容槽位。
- 案例详情会按项目模板显示素材缺口和账号健康提示。
- 异常账号会显示运营建议，把断更、待核对、素材缺口、播放下滑转成下一步动作。
- 交付包生成后，可以在排期卡片里直接「打开交付包」。
- 候选稿卡片可直接复制执行说明和抖音发布文案，方便发微信。
- 排期卡片可以标记异常，避免卡在无法推进的内容上。
- 素材扫描后，可以把素材标记为「可用 / 需处理 / 不可用」。
- 素材缺口旁可以直接创建对应 Image 任务，也可以复制补素材说明发给素材整理人员。
- 核对队列里点「核对并回填」可录入粉丝、播放、点赞、评论。
- 「爆款链接」页只要求工作人员粘贴抖音作品链接；系统先记录为待分析，首页也会显示待分析链接，后续由 Codex/AI 用青豆提取、拆结构，再按适合/禁用人设关键词批量生成。
- 待爆款分析链接可以复制青豆任务，任务文本包含链接、提取字段、回填标准和下一步。
- Image 任务可以打开输出目录、复制 Prompt，并标记 `review / approved / rejected`。
- 剪辑任务会生成 `剪辑任务单.txt`，可复制给剪辑人员，也可打开本地任务目录，并在首页处理待剪辑状态。
- 交付包会生成 `05-素材顺序清单.txt`，按顺序说明微信要发送的图片或视频。
- 视频/口播交付包会额外生成 `03-口播字幕文案.txt`、`04-剪辑要求.txt` 和 `07-成片回收说明.txt`。
- Image 任务会在输出目录生成 `图片任务说明.txt`，包含 prompt、negative prompt、用途和保存目录。
- 待核对任务可以复制抖音核对清单，按发布状态、发布时间、标题正文、素材顺序、数据和评论区问题逐项检查。
